About the website


Search mechanism:

The search works in 2 phases, a tag phase and a filename phase.
  1. Comics consist of a number of tags.
  2. If a word in the search is a tag; it is added
  3. If a word in search is a word in the filename; it is added
Some comics are restricted and will not be shown unless specific search terms are used

List of comic tags

  • Date: Each comic's date of release also acts as a tag. Format (YYYYMMDD)
  • Over100: Comics over 100 pages long
